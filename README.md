# NaïveProxy Manager 

Интерактивный установщик и менеджер NaïveProxy

## Установка
```bash
sudo curl -fsSL https://raw.githubusercontent.com/pumbaX/naiv/main/NaiveProxy.sh -o NaiveProxy.sh && sudo bash NaiveProxy.sh
``` 

## Требования
- Ubuntu 22/24, Debian 11/12
- root
- домен с A-записью на IP сервера
- свободные порты 80 и 443
